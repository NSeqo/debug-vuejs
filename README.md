### 组件的事件监听处理

1. 事件监听器 {name: "click", once: false, capture: false, passive: false}， 注册事件的时候passive为true，那么监听器函数内的preventDefault() 将会被浏览器忽略

2. 事件处理函数会被存在vm._events数组中
